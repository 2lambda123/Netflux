exportODE

PURPOSE

Exports system of equations to standalone .m file.

SYNOPSIS

function [commandLine] = exportODE(speciesNames,paramList,ODElist)

DESCRIPTION 

Exports system of equations to standalone .m file.
 
   exportODE This function exports the system of differential equations as a
   standalone .m file. Different from exportODE2 in that exportODE writes a
   call to a solver (ode23) while exportODE2 does not.  Input: paramsList =
   parameter list; speciesNames = list of speciesNames; ODElist = list of
   ODEs Output: commandLine = cell that holds everything Version 0.08a,
   08/31/2011 by JJS
