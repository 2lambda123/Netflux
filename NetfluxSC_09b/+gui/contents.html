DESCRIPTION 

 EasyGUI: A package for creating GUIs quickly and easily
 
 GUI controls (aka. widgets)  
    gui.checkbox      - check box
    gui.edittext      - field for entering text 
    gui.editnumber    - field for entering numbers
    gui.listbox       - scrollable list 
    gui.textmenu      - drop-down menu 
    gui.numericmenu   - drop-down menu of numbers
    gui.slider        - slider for choosing number in a range
    gui.pushbutton    - simple button
    gui.togglebutton  - button that stays pushed in or out
    gui.label         - text label with formatting
    gui.space         - add horizontal and vertical space

    gui.daqinput      - configure and acquire analog data
    gui.stripchart    - scrolling plot
    gui.group         - group a set of widgets

 Layout of widgets
    gui.autogui       - figure with automatic layout of widgets
    gui.manualgui     - figure with manual layout of widgets

 Miscellaneous
    gui.version       - shows the version of EasyGUI
    gui.monitor       - monitor a set of widgets for inputs

 Examples (look under examples/)
    catalog          - Displays an interactive catalog of EasyGUI widgets

    autodemo         - Simple GUI with automatic layout
    autodemo2        - Simple GUI, input processed with a single callback
    autodemo3        - Simple GUI, input processed with several callbacks
    manualdemo       - Simple GUI with manual layout
    dialog_demo      - A sample dialog box created using EasyGUI
    sharedata_demo   - A GUI that gets user input from a dialog box

    fitznag         - GUI with ODE simulation [Fitzhugh-Nagumo model]
    lorenzgui       - GUI with ODE simulation [Lorenz chaotic attractor]
    fftsmooth       - GUI with interactive image filtering
    bpfilter        - Interactive bandpass filter 
    rlc_circuit     - Interactive RLC circuit circuit simulation

    daqplot         - Analog data acquisition GUI
    daqplot2        - Data acquisition GUI with stripchart
    daqplot3        - Data acquisition GUI that uses a callback
    daqfft          - Data acquisition GUI with analysis and visualization
